# Отчет по лабораторной работе №12

## Детальное описание выполнения работы:

**Шаг 1: Установка Git**
- Скачал установщик с gitforwindows.org
- Запустил установку с настройками по умолчанию
- Проверил установку командой git --version

**Шаг 2: Создание локального репозитория**
- Создал папку homework_IIT
- Выполнил git init для инициализации репозитория
- Настроил пользователя: git config --global user.name и user.email

**Шаг 3: Создание файла отчета и первый коммит**
- Создал файл README.md с начальным описанием
- Выполнил git add README.md для добавления в отслеживание
- Создал первый коммит: git commit -m \"Создан файл отчета\"

**Шаг 4: Работа с ветками**
- Создал новую ветку: git branch detailed-report
- Переключился на ветку: git checkout detailed-report
- Добавил детальное описание шагов в README.md
- Зафиксировал изменения в новой ветке

**Шаг 5: Слияние веток**
- Вернулся в основную ветку: git checkout main
- Выполнил слияние: git merge detailed-report
- Проверил историю коммитов: git log --oneline

**Шаг 6: Работа с GitHub**
- Создал репозиторий homework-IIT на GitHub
- Связал локальный репозиторий с удаленным: git remote add origin
- Загрузил код на GitHub: git push -u origin main

**Шаг 7: Публикация статического сайта**
- Создал папку website для статического сайта
- Добавил HTML файлы сайта
- Настроил GitHub Pages для публикации сайта

## Вывод
Лабораторная работа успешно завершена. Освоены основные операции Git: создание репозитория, работа с ветками, слияние, работа с удаленным репозиторием на GitHub и публикация статического сайта.
